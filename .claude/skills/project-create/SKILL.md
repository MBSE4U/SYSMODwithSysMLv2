---
name: project-create
description: Generates a SYSMOD project.

Use when the user requests creating a new SYSMOD project. 
---

When generating a SYSMOD project:

Create the SysML v2 textual notation for the project as specified in the SYSMOD SysML v2 library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml. The AI metadata of the AIProject element in the SYSMOD library provides a create_prompt. Use that prompt.
