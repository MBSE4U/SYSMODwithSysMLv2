---
name: stakeholder-create
description: Generates stakeholders based on SYSMOD ExtendedStakeholder.

Use when the user requests creating new SYSMOD stakeholders. 
---

When generating a stakeholders:

Create the SysML v2 textual notation for the stakeholders as specified in the SYSMOD SysML v2 library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml. The stakeholderAI metadata of the AIProject element in the SYSMOD library provides a create_prompt. Use that prompt.
