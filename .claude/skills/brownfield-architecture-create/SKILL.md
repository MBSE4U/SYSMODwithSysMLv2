---
name: brownfield-architecture-create
description: Generates a SYSMOD brownfield-architecture.

Use when the user requests creating a new SYSMOD brownfield-architecture. 
---

When generating a SYSMOD brownfield-architecture:

Create the SysML v2 textual notation for the brownfield-architecture as specified in the SYSMOD SysML v2 library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml. The brownfieldContextAI metadata of the AIProject element in the SYSMOD library provides a create_prompt. Use that prompt.
