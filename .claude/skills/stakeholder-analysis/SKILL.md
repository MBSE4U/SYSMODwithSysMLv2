---
name: stakeholder-analysis
description: Generates a stakeholder priority map based on SYSMOD stakeholders.

Use when the user requests a stakeholder priority map.
---

When creating a stakeholder priority map:

Create a two-dimensional map with risk as the y axis and effort as the x axis. Locate the stakeholders exposed in the prompt according to their risk and effort property values in the map. Each stakeholder is depicted by a stakeholder symbol with its name. The stakeholder are defined by the SYSMOD ExtendedStakeholder which is defined in the SysML v2 SYSMOD library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml.
